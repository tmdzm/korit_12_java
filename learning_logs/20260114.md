# JSON
## 정의

- Json(JavaScript Onject Notation) : 본래 JavaScript 언어로부터 파생되어 JavaScript구문을 따르지만 언어 독립형 데이터 포맷
- 이상에서 말한 독립적이라는 특성 때문에 JS 뿐만 아니라 Java / python 등 다양한 언어에서 데이터를 주고 받는 포맷에 해당한다.
