
# ch03 Scanner
- scanner를 쓰기위해선
1. Scanner를 import하고
2. scanner 객체를 생성한뒤
3. 메서드 호출을 이용해 사용한다.

```java
Scanner sc = new Scanner(System.in);
```

- 메서드는 크게 nextLine()과 아닌것으로 나눌 수 있다.
- nextLine()은 개행문자(\n)까지 읽기때문에 띄어써도 읽지만, 나머진 아니다.
- 개행을 읽기때문에 nextLine()전에 다른 next()로 입력했다면, \n를 읽고 바로 아무것도 입력받지 않을 수 있기때문에, 순서를 바꾸거나 nextLine()을 하나 더 쓰는 편법이 필요하다.

```java
        System.out.println("올해는 몇 년도 인가요? >>>");
        int year = sc.nextInt();
        sc.nextLine();//\n를 한번 읽는 꼼수,변수에 대입할 필요가 없다.
        System.out.println("니 이름이 뭔데.");
        String name2 = sc.nextLine();
        System.out.println("연도 : " + year);
        System.out.println("이름 : " + name2);

```


# ch04 Conversion
### 형변환
- 크게 형변환은 업캐스팅과 다운캐스팅 2가지가 있다.
- 업캐스팅의 경우, 더 큰 용량의 자료형으로 변환하므로, 암시적 변환이 가능하다.
- 다운캐스팅의 경우, 더 작은 용량의 자료형으로 변환하므로, 명시적 변환이 가능하다.

```java
//업캐스팅, 명시하지 않아도 된다.
        char cast3 = 'b';
        int casted3 = cast3;
//다운캐스팅
        int cast4 = 99;
        char casted4 = (char)cast4;//char임을 명시
```

# ch05 Condition
### 조건문
- 주로 if와 else if와 else를 이용, 조건식이 참일때 실행문이 실행한다.
- 조건식은 true또는 false로 값이 나오는 식을 말한다.
```java
    if(조건식){
        실행문
    }
```

- 삼항연산자 
```java
    i = a < b ? a : b;
    //i의 값은 a가 b보다 작을때(참일때) a, 아닐때 b
```
