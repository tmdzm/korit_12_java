# 참조 자료형 캐스팅
- 객체 지향 프로그래밍에서 중요한 요소 중 하나

## 사전 지식
    - java 상에서의 자료형은 크게 두가지로 구분된다.
        1. 기본 자료형 : int, double, char등
        2. 참조 자료형 : 객체를 참조하는 변수
    - 참조 자료형 캐스팅이란 - 객체의 자료형을 변환하는 작업
    - int -> double / int -> long 으로 바뀌는건 형변환(conversion)

## 캐스팅 종류
    1. 업캐스팅 : 하위 클래스 타입의 객체를 상위 클래스 타입으로 변환
    2. 다운캐스팅 : 상위 -> 하위

## 미리 기억해 두면 좋은 연산자(operator) : instanceof
    - 1 + 1 == 2라고 했을때 +가 연산자
    - 객체명 instanceof 클래스명 -> true // false 가 나온다.

## 업캐스팅
    - 암시적으로 이루어짐

## 다운캐스팅
    - 명시적으로 이루어짐

## 업캐스팅과 다운캐스팅의 활용
    - 캐스팅은 다형성(Ploymorphism)을 구현할 때 사용. 상위 클래스 타입으로 객체를 처리하면서도 특정 상황에서는 하위 클래스의 고유 기능을 사용 가능
    -> 클래스 타입을 전환시크는게 중요한 이유는 자기가 만든 클래스의 객체를 기존 개발자들이 만든 클래수로 전환시킬 수 있다면 이미 만들어진 기능을 손쉽게 쓸 수 있기 때문
