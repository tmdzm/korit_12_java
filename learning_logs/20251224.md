# Method
## 정의

- function vs method
        그렇다면 왜 용어를 분리하는가
- 기본적으로 클래스 내부에 함수를 정의하게 되면 이를 '메서드'라고 한다.
JAVA는 모든 함수들이 특정 클래스 내부에 있다.
그래서 다른 언어를 배우게 된다면 '함수의 호출 방식'과 '메서드'의 호출방식'상에서의 차이가 존재하지만, 현재 수업을 기준으론 전부다 method에 해당, Java수업중 함수는 전부 메소드라 생각하라

## 용어정리
1. method 정의 : 사용자가 메소드를 새로 만드는것
2. method 호출 : 만들어진 메소드를 실행시키는것
3. 매개변수(parameter) : 특정 메소드에 필수적으로 요구되어지는 변수
4. 출력 / 결과/ 리턴값 : 메소드 호출의 출력값-아웃풋
5. 인수 / 인자(argument) : 메소드에 전달하는 입력값

## java 상에서의 method
'사진을 찍는다'를 예시로 들자면
1. 주머니에서 폰을 꺼낸다.
2. 잠금화면을 푼다
3. 카메라를 켠다.
4. 상에 조준한다
5. 찰칵

사실 현실에선 '찰칵'만 하면 되지만
컴퓨터에선 5개의 순서를 일일이 만들어야한다
이를 도와주는것이 method다.
method로 일련의 행위를 묶어 호출한번에 모두 일어나게 할 수 있다.
